模型放置说明
----------------

请准备并放置以下文件到固件的 root 或 romfs 路径下（示例路径 `/rom/models/`）：

- `face_det.nbg`         : 人脸检测模型（建议输入尺寸例如 640x640，输出边界框）。
- `face_68_kpt.nbg`     : 面部 68 点关键点模型（输入为人脸裁剪图像，例如 256x256，输出 68 个 (x,y) 规范化坐标）。

测试图像：
- `test.argb`           : 用于本地测试的原始 ARGB8888 raw 文件（4 字节/pixel，A R G B）。可选。

注意：
- `.nbg` 为厂商 NNA 专用模型格式；如果你有 tflite/onnx 模型，需要通过厂商提供的工具链或转换工具导出为 `.nbg`。
- 模型的输出格式（尤其关键点模型）必须与示例代码中的解析逻辑一致；否则需在 `src/application.c` 中实现对应的后处理解析。
